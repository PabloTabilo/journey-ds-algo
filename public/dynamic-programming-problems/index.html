    Dynamic Programming Problems #

    To understand this type of problem, the first step is to identified the characteristics. For example:

    • It’s a maximum / minimum problem?
    • Do I need to count a number of events?
    • Can this problem be solve using recursion?
    • Does the solution follow the same properties or structure, independent of the problem’s size?
    • In your implementation on the recursive function, is there a specific order require to process the solution?
